1️⃣ 准备工作

你需要：

  • 一台电脑（Mac / Linux / Windows 都可以）或一台云服务器（VPS）
  • Node.js 22 或更高版本
  • 一个 AI 模型的 API Key（下面会教你怎么拿）
如果你只想在自己电脑上用，Mac/Linux 直接装就行。如果想 24 小时在线，建议用云服务器（VPS）。

2️⃣ 获取 AI API Key（最重要！）

OpenClaw 本身是免费的，但它需要连接一个 AI 模型来"思考"。你需要从 AI 公司获取一个 API Key（类似密码）。

以下三个选一个就行，推荐 Anthropic Claude：

🟣 Anthropic Claude（推荐）

▶
1

打开 Anthropic Console

打开 console.anthropic.com →

如果没有账号，点右上角 Sign Up 注册（支持 Google 账号登录）

2

进入 API Keys 页面

导航路径
左侧菜单 → API Keys

或者直接打开：

直接打开 API Keys 页面 →
3

创建新的 API Key

操作步骤
  1. 点击右上角 「+ Create Key」 按钮
  2. 给 Key 起个名字（比如 "openclaw"）
  3. 选择 Workspace（默认就行）
  4. 点击 「Create Key」
🔑 生成的 Key 长这样：sk-ant-api03-xxxxxxxxxxxxx
⚠️ 这个 Key 只会显示一次！立刻复制保存！
4

充值（必须）

新账号有 $5 免费额度。用完后需要充值：

左侧菜单 → Plans & Billing → Add Payment Method
打开充值页面 →

💰 日常聊天大约 $5-10/月，很便宜

推荐模型：claude-sonnet-4-20250514（性价比最高）或 claude-opus-4-0（最强）

充值

打开充值页面 →
推荐模型：gpt-4o（最强）或 gpt-4o-mini（便宜）

💎 Claude Max 订阅（不需要 API Key）

▶

3️⃣ 安装 OpenClaw

🚀

一键脚本

最简单

适合个人电脑

🐳

Docker

推荐服务器

适合 VPS / 24小时运行

方式一：一键脚本（个人电脑）

打开终端（Terminal），粘贴以下命令：

Mac / Linux：

curl -fsSL https://openclaw.ai/install.sh | bash

Windows（PowerShell）：

iwr -useb https://openclaw.ai/install.ps1 | iex

安装过程中会问你几个问题：

  1. 选择 AI 提供商 → 选 Anthropic / OpenAI / Google（取决于你上面拿了哪个 Key）
  2. 输入 API Key → 粘贴你刚才复制的 Key
  3. 是否安装为系统服务 → 选 Yes（这样重启电脑后也会自动运行）
安装完成后，打开浏览器访问 http://127.0.0.1:18789/ 就能直接聊天了！

方式二：Docker（服务器 / VPS）

适合在云服务器上 24 小时运行：

# 1. 克隆代码
git clone https://github.com/openclaw/openclaw.git
cd openclaw

# 2. 用预构建镜像安装（更快）
export OPENCLAW_IMAGE="ghcr.io/openclaw/openclaw:latest"

# 3. 运行安装脚本
./docker-setup.sh

脚本会引导你完成配置，和一键脚本类似。

VPS 至少需要 2GB 内存。如果内存不够，先加个 swap：
sudo fallocate -l 2G /swapfile && sudo chmod 600 /swapfile && sudo mkswap /swapfile && sudo swapon /swapfile

4️⃣ 连接 Telegram

1

在 Telegram 创建 Bot

  1. 打开 Telegram，搜索 @BotFather
  2. 发送 /newbot
  3. 输入 bot 的显示名称（比如 "我的AI助手"）
  4. 输入 bot 的用户名（必须以 bot 结尾，比如 myai_helper_bot）
  5. BotFather 会回复一个 token，格式：123456789:ABCdef...
复制这个 token！下一步要用。
打开 BotFather →
2

在 OpenClaw 中添加 Telegram

openclaw channels add --channel telegram --token "粘贴你的BOT_TOKEN"

然后重启网关：

openclaw gateway restart
3

配对你的账号

  1. 在 Telegram 中找到你刚创建的 bot，给它发一条消息（比如 "hi"）
  2. 回到终端，运行：
# 查看配对请求
openclaw pairing list telegram

# 批准配对（把 CODE 换成显示的配对码）
openclaw pairing approve telegram CODE

✅ 配对成功后，你就可以通过 Telegram 和 AI 聊天了！

5️⃣ 连接 WhatsApp

1

扫码连接

openclaw channels login
  1. 终端会显示一个 QR 码
  2. 打开手机 WhatsApp → 设置 → 已关联设备 → 关联设备
  3. 扫描终端中的 QR 码

✅ 连接成功后，别人给你发 WhatsApp 消息，AI 就会自动回复！

WhatsApp 连接是绑定你的个人账号的。如果你退出 WhatsApp 或换手机，需要重新扫码。

6️⃣ 日常使用

常用命令

# 查看运行状态
openclaw status

# 查看日志（实时）
openclaw logs --follow

# 重启服务
openclaw gateway restart

# 打开浏览器控制台
openclaw dashboard

# 检查配置问题
openclaw doctor

在哪里聊天？

  • 🌐 浏览器：打开 http://127.0.0.1:18789/
  • 📱 Telegram：直接给你的 bot 发消息
  • 📱 WhatsApp：给自己发消息（或别人发给你）

7️⃣ 常见问题

❓ 输入 openclaw 命令提示"找不到"

▶

需要把 npm 的全局目录加到 PATH：

# Mac/Linux：加到 ~/.zshrc 或 ~/.bashrc
export PATH="$(npm prefix -g)/bin:$PATH"

# 然后重新打开终端

❓ Telegram bot 不回复消息

▶
  1. 检查 OpenClaw 是否在运行：openclaw status
  2. 检查是否已配对：openclaw pairing list telegram
  3. 查看日志找原因：openclaw logs --follow
  4. 确认 bot token 正确

❓ API 提示余额不足

▶

需要去对应平台充值：

  • Anthropic：充值页面
  • OpenAI：充值页面
  • Google Gemini：免费额度用完后需要启用付费计划

❓ 怎么更新 OpenClaw？

▶ 
# npm 安装的
npm update -g openclaw
openclaw gateway restart

# Docker 安装的
cd openclaw && git pull
docker compose down
export OPENCLAW_IMAGE="ghcr.io/openclaw/openclaw:latest"
./docker-setup.sh
